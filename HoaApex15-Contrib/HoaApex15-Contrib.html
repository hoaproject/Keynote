We love contributions

Hoa is the sum of the expertise and experience of its contributors.

More contributions means:

  • New features,
  • More bug fixed,
  • Better support for users,
  • Better documentation.

Our new contribution process aims to keep our code & docs quality standard high.

But what can I do?

Each library has now its board to centralize all works and its progression:

  • Backlog, things under discussion,
  • Ready, things that are ready to be done,
  • In progress, things that are being done by some contributors,
  • Done, things that have been recently done.

Central has its own board to provide the highest overview of work around Hoa.

The task that fit you like a glove

Most of the works on a board have a difficulty label:

  • casual, can be done by a new contributor or can be done quickly,
  • medium, requires some skills to address this work,
  • hard, requires strong skills in the code and the domain it addresses.

It is really innovative for an open source project !

Preparing your patch...

Your patch must follow these criteria:

  • Follow PSR-1 and PSR-2 coding style,
  • Doesn't break the existing tests,
  • Bonus: add tests,
  • Bonus: add documentation.

We are very demanding on patch quality but...

...we have a shiny new toolbox

It includes everything to help you prepare the perfect patch:

  • devtools:cs,
  • test:run,
  • devtools:documentation.

Writting a commit message like a boss

It must be written in english and its body must be formatted using Markdown.

It must answer to the following questions:

  • What was the issue?
  • How to address this issue?
  • How did we address the issue?

Reviewers are your best friends

They are assigned to your pull request and are here to:

  • Guide you,
  • Help you,
  • Give you pointers and links to useful resources.

Note: Being a reviewer is also a kind of contribution!

Conclusion

We are using a classical workflow (fork, pull request) with high quality standards.

We provide a toolbox and awesome humans (reviewers) to help you contribute to Hoa !

Thank you !
